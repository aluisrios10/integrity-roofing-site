## Project Overview

The website provides a modern, responsive interface for roofing customers. It includes:

- Service descriptions  
- Online quote requests  
- Appointment scheduling  
- Contact information with map integration  

The design prioritizes clarity, trust, and user experience.

---

## Features

- **Responsive Layout** – Works on desktop, tablet, and mobile.  
- **Contact & Quote Forms** – Submissions integrated with email notifications.  
- **SEO Optimized** – Metadata, Open Graph, and Twitter Card tags included.  
- **Dynamic Page Titles** – Each page can have a custom title for SEO and usability.  
- **Favicon Support** – Properly linked favicon for branding.
